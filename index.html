Introduction to Database Systems

This course provides an introduction to the fundamentals of database systems, including data modeling, relational databases, query languages, and data storage and retrieval. Students will learn how to design and implement a database system to store and manage data efficiently.

Database Security

This course covers the principles of database security, including authentication, authorization, encryption, auditing, and access control. Students will learn how to protect their databases from unauthorized access and malicious attacks.

Advanced Database Design

This course focuses on advanced topics in database design such as normalization, indexing, query optimization, transaction processing, distributed databases, object-oriented databases and data warehousing. Students will gain an understanding of how to design a robust database system that meets the needs of their organization.
